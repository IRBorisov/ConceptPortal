# Конституенты

Читай перед `applySchemaPatch`.

## Поля

- `id` — стабильный id внутри сессии.
- `alias` — имя вида `X1`, `D5`, `F2`; префикс должен совпадать с `cstType`.
- `cstType` — роль конституенты.
- `definitionFormal` — формальное определение; для `basic` и `constant` строго `''`; для `structure` — область определения.
- `term` — термин в естественном языке, используется в текстовых определениях. Если обозначаются элементы множества, то пишется с маленькой буквы во множественном числе. Иначе с большой буквы в единственном.
- `definitionText` — текстовое определение, соответствующее формальному определению. Для простых выражений не указывается если совпадает с термином. Каждый используемый термин должен быть введен в схеме.
- `convention` — соглашение для неопределяемого понятия. Может использовать термины, не определенные в схеме.

Пропущенные текстовые поля сохраняются как `''`. Пиши `term`, `definitionText`, `convention` на одном языке: в языке схемы или, для новой схемы, в языке запроса.

Не добавляй точку в конце текстовых полей.

## `cstType`

- `nominal` / `N#` — номиноид. В родоструктурной экспликации и формальных определениях не используется.
- `basic` / `X#` — базисное множество; формула пустая. Без арифметики и порядка над элементами.
- `constant` / `C#` — константное множество; формула пустая. С арифметикой и порядком; интерпретация в КМ. См. [BASE-SELECTION.md](BASE-SELECTION.md) (отличие от `X#` и встроенного `Z`).
- `structure` / `S#` — родовая структура; `definitionFormal` задает область определения (типизацию).
- `term` / `D#` — терм; формальное определение задает вычисляемое значение.
- `function` / `F#` — терм-функция. Для вычисляемой зависимости «аргумент → значение» предпочитай `F#` множеству пар.
- `predicate` / `P#` — предикат-функция.
- `axiom` / `A#` — аксиома; нужна типизация `Logic`.
- `statement` / `T#` — высказывание; нужна типизация `Logic`.

В модели `basic`, `constant`, `structure` можно интерпретировать через `setModelValues`. `term`, `function`, `predicate`, `axiom`, `statement` вычисляются и напрямую не задаются.

## Валидация

1. Префикс `alias` соответствует `cstType`.
2. У `basic` и `constant` пустая формула, иначе `definitionNotAllowed` (`0x8862`).
3. Все глобальные ссылки на **конституенты** (`X#`, `C#`, `S#`, `D#`, `F#`, `P#`, …) уже есть в сессии. `R#` — радикалы, не конституенты; записи с `alias` `R1`, `R2`, … в схеме быть не должно ([SYNTAX.md](SYNTAX.md), раздел «Радикалы»).
4. `axiom` и `statement` имеют типизацию `Logic`, иначе `expectedLogic`.
5. Выводимая конституента с пустой формулой дает `cstEmptyDerived`.
